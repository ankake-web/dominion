# 第二版＋プロモ 追加カード — 実装リファレンス

このゲームは当初「初版（First Edition）」構成だったが、**基本・陰謀を第二版（Second Edition）に更新**し、さらに**プロモカード**を追加した。
初版で廃止されたカードは実装を残し、セット選択の「初版・王国基本／初版・陰謀」で遊べる。

## セット選択（js/cards.js の DOM.CARD_SETS）
- **王国基本セット（第二版）/ 陰謀セット（第二版）** … 標準（デフォルト）。
- **おすすめ** … テーマ別の固定10種。
- **初版・王国基本 / 初版・陰謀** … 第二版で廃止された懐かしのカード入り。
- **プロモ全部入り** … プロモ6種＋基本4種。
- **ランダム** … 基本＋陰謀／陰謀のみ／基本のみ／プロモ込み／初版から。

## 第二版での変更
- **基本：廃止6種** 木こり・宰相・祝宴・冒険者・密偵・泥棒（実装は残す）。
- **基本：追加7種** 前駆者・商人・家臣・密猟者・山賊・衛兵・職人。
- **陰謀：廃止6種** 大広間・銅細工師・破壊工作員・斥候・秘密の小部屋・貢物（実装は残す）。
- **陰謀：追加7種** 廷臣・外交官・待ち伏せ・風車・パトロール・身代わり・隠し通路。

## 追加カードの効果（正典）
### 基本 第二版
- **前駆者 (Harbinger, $3)**：+1カード+1アクション。捨て札から1枚を山札の上に置いてよい。
- **商人 (Merchant, $3)**：+1カード+1アクション。このターン最初に銀貨を出したとき+1コイン（商人の数だけ）。
- **家臣 (Vassal, $3)**：+2コイン。山札の上を1枚捨て、それがアクションなら使ってよい（アクションは消費しない）。
- **密猟者 (Poacher, $4)**：+1カード+1アクション+1コイン。空のサプライ1つにつき手札を1枚捨てる。
- **山賊 (Bandit, $5, アタック)**：金貨を獲得。他の各人は上2枚を公開し銅貨以外の財宝1枚を廃棄、残りを捨てる。
- **衛兵 (Sentry, $5)**：+1カード+1アクション。山札の上2枚を見て、好きな数を廃棄／捨て、残りを好きな順で上に戻す。
- **職人 (Artisan, $6)**：コスト5以下を手札に獲得し、手札1枚を山札の上に置く。

### 陰謀 第二版
- **廷臣 (Courtier, $5)**：手札1枚を公開。その種類数だけ次から選ぶ（異なるもの）：+1アクション／+1購入／+3コイン／金貨獲得。
- **外交官 (Diplomat, $4, リアクション)**：+2カード。引いた後手札5枚以下なら+2アクション。リアクション：他人のアタック時、手札5枚以上で公開→2枚引き3枚捨てる（無効化はしない）。
- **待ち伏せ (Lurker, $2)**：+1アクション。サプライのアクションを廃棄、または廃棄置場からアクションを獲得（択一）。
- **風車 (Mill, $4, 勝利点1)**：+1カード+1アクション。手札2枚を捨てれば+2コイン（任意）。
- **パトロール (Patrol, $5)**：+3カード。上4枚を公開し勝利点と呪いを手札に、残りを好きな順で上へ。
- **身代わり (Replace, $5, アタック)**：手札1枚を廃棄→最大$2高いを獲得。ア／財は山札の上、勝利点なら他全員が呪いを獲得。
- **隠し通路 (Secret Passage, $4)**：+2カード+1アクション。手札1枚を山札の好きな位置に入れる。

### プロモ
- **城壁のある村 (Walled Village, $4)**：+1カード+2アクション。クリーンアップ開始時、場のアクションが（自身含め）2枚以下なら山札の上に戻す（本実装は自動）。
- **使者 (Envoy, $4)**：上5枚を公開。左隣が1枚選んで捨てさせ、残りを手札に。
- **総督 (Governor, $5)**：+1アクション。全員に効果（自分はカッコ内の強い方）：全員+1（+3）カード／全員 銀貨（金貨）／全員 任意で廃棄しちょうど$1（$2）高いを獲得。
- **取り壊し (Dismantle, $4)**：手札1枚を廃棄。コスト$1以上なら、それより安いカードと金貨を獲得。
- **闇市場 (Black Market, $3)**：+2コイン。闇市場デッキ（サプライ外の王国カードで構成）の上3枚を公開。手札の財宝を出してよい。1枚を購入してよい（購入回数は消費しない）。残りは底へ。
- **隠し財産 (Hoard, $5, 財宝)**：コイン+2。場にある間、勝利点カードを購入すると金貨を獲得。
  - ※「隠し財産」は本来 拡張『繁栄(Prosperity)』のカード（純正プロモのStashはシャッフル時の任意配置がこのエンジンに馴染まないため不採用）。基本＋陰謀だけで成立し、名前が一致するため採用した。

## カードを1枚足すとき触る層
> 表示データとオンライン許可リストは**単一ソース化＆自動検証**済み（[[整合性テスト]]）。
> 二重管理が減り、抜けは `test/integrity.test.js` が即検出する。

1. **`js/cards.js`（正本・必須）** … `DOM.CARDS` 定義（id/name/cost/types/coin/vp/text）と、抽選母集団に出すなら `DOM.POOLS`／固定セットなら `DOM.CARD_SETS` に id を追加。**名前・コスト・種別はここだけ**（昔のように cards.json と二重に書かない）。
2. `js/engine.js` … 効果なら `applyEffect()` の case、選択を伴うなら `reduce()` に `*_RESOLVE` 系 case と補助関数。**新しいアクション種別は同じ engine.js の `PLAYER_ACTIONS` にも足す**（サーバ許可は自動。整合性テストが switch と一致を検証）。特殊フックは `initSupply`（勝利点の山）/`cleanupAndAdvance`（城壁のある村）/`createInitialState`＋`maskStateFor`（闇市場デッキ）を参考に。
   - **アタックカードは `ATTACKS` 登録表に1行足す**（＋ `xxxEnterVictim` と `*_REACT` リゾルバ）。堀／秘密の小部屋／外交官の反応窓口と「無効化されたら次の被害者へ」は登録表から自動処理されるので、`MOAT_REVEAL` 等を個別に触らない。登録忘れは整合性テスト＋「堀で全アタック無効化」テストが検出する。
   - **定型の選択は共通部品を使う**（コピペ削減）：`discardFromHand(state, 席, cards, 枚数, ログ)`／`trashFromHand(...)`／`finishGain(state, pd, card, canGain, dest, ログ)`。特に `finishGain` は“強制獲得（候補があれば辞退不可・尽きたら辞退OK・在庫/条件チェック）”を1か所に集約済み。獲得系リゾルバはこれを呼んで `return state` するだけ。
3. `js/cpu.js` … `chooseAction()` の優先順、`decidePending()` の case、`GAIN_ORDER`（**全カードを含む必要あり**＝整合性テストが検証）。
4. `js/ui.js` … `viewPendingModal()` の分岐＋必要な `modal*` ヘルパ。
5. （任意）`js/carddata.js` の `DISPLAY` に icon・effects … プレビュー(cards.html)の見栄え用。**省略可**（無ければ種別アイコン＋text から自動表示）。表示データ自体は `DOM.CARDS` から自動導出。
6. （任意）画像 `asset/<id>.jpg`（未配置でも文字カードにフォールバック）。
7. テスト `test/edition2.test.js`（エンジン）／`test/edition2-ui.test.js`（UI）。`test/integrity.test.js` が構造の抜け（許可リスト・GAIN_ORDER・プール・表示データ）を自動でガード。

> サーバ `server/gameServer.js` は `DOM.engine.PLAYER_ACTIONS` をそのまま許可リストに使うので**触らなくてよい**（以前のように `ALLOWED` へ手動追加する必要なし）。

## 修正したバグ
- **勝利点の山の枚数**：王国の勝利点カード（庭園・公爵・貴族・後宮・大広間・風車）も屋敷/公領/属州と同じく **2人=8枚 / 3-4人=12枚** にした（以前は一律10枚）。
- **詐欺師のコスト**：正しくは **$3**（以前は誤って$5）。
