# 基本セット（第1版）残り15種 — デザイン/実装リスト

実装済み10種（cellar/village/woodcutter/workshop/moat/militia/smithy/remodel/market/mine）は第1版「基本」構成。
第1版「基本」の王国カードは全25種なので、**残り15種**が未実装。

## 必要な画像（デザイン担当が用意）
各カードにつき `asset/<id>.jpg`（原寸）と `asset/thumb/<id>.jpg`（サムネ）。未配置でもカード名テキスト表示にフォールバックする。

## 補足（実装時の要点）
- `gardens` は得点が変動（デッキ枚数依存）。engine の `vpOf()` に公爵(`duke`)と同じく特例加算を1か所足す（`Math.floor(総枚数/10)`）。
- `witch`/`bureaucrat`/`spy`/`thief` はアタック。堀(`MOAT_REVEAL`)の対象 pd.type に追加が必要。
- `throne_room` は「アクションをもう一度使う」再入処理が必要で最難。pending を伴うアクション（民兵・拷問人など）との二重起動の整合に注意。実装は最後に回すのが安全。
- `council_room` は他プレイヤーに1枚ずつ引かせる（手番に関係ない他者ドロー）。

実装難易度の推奨順：研究所 → 祝祭 → 金貸し → 宰相 → 魔女 → 礼拝堂 → 庭園 → 祝宴 → 冒険者 → 議会 → 役人 → 密偵 → 書庫 → 泥棒 → 玉座の間。
